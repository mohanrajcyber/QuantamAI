## 🐛 If Something Doesn't Work

### Issue 1: No videos appear
**Check:**
- Internet connection
- Browser console (F12)
- API key in `src/app/services/youtubeService.ts`

**Solution:**
- Refresh page (F5)
- Try different search term
- Check console for errors

### Issue 2: "Quota exceeded" error
**Reason:**
- Used 100 searches today

**Solution:**
- Wait until tomorrow (resets daily)
- Or create new Google Cloud project

### Issue 3: Videos not playing
**Check:**
- Video is not restricted
- Internet connection
- Try different video

**Solution:**
- Click different video
- Refresh page
- Check internet

## 📊 API Quota Info

**Free Tier:**
- 100 searches per day
- Unlimited video plays
- Resets daily at midnight PST

**Usage:**
- Each search = 1 search
- Playing video = FREE
- Loading trending = FREE (1 unit only)

**Example:**
- Search "lofi music" = 1 search

- Search "coding music" = 1 search
