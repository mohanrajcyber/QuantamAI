## Troubleshooting

### Login page not showing?
Open: `http://localhost:5173?logout=force`

### Servers not starting?
1. Close all Node.js processes
2. Run: `RESTART-ALL-SERVERS.bat`

### Backend errors?
Check: `backend/.env` file exists

### Frontend errors?
Run: `npm install` in root folder
