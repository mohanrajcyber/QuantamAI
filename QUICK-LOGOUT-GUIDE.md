# Quick Logout Guide - Quantum AI

## Problem: Login Page Not Showing?

You're automatically logged in because you used "Skip Login" before.

## FASTEST FIX (1 Second!)

Just open this URL in your browser:

```
http://localhost:5173?logout=force
```

**That's it!** The login page will appear instantly.

---

## Other Methods

### Method 1: Browser Console
1. Press `F12`
2. Type: `localStorage.clear()`
3. Press Enter
4. Refresh page (F5)

### Method 2: Clear Browser Data
1. Press `Ctrl + Shift + Delete`
2. Clear "Cookies and site data"
3. Refresh page

### Method 3: Use Logout Button
1. Click user icon (top right)
2. Click "Logout"
