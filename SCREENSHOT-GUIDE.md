## 📝 How to Take Screenshots

### Method 1: Windows Snipping Tool (Recommended)
```
1. Press: Windows + Shift + S
2. Select area to capture
3. Screenshot copied to clipboard
4. Open Paint
5. Paste (Ctrl + V)
6. Save as PNG in screenshots folder
```

### Method 2: Full Screen
```
1. Press: PrtScn (Print Screen)
2. Open Paint
3. Paste (Ctrl + V)
4. Crop if needed
5. Save as PNG
```

### Method 3: Browser Extension
```
1. Install: Awesome Screenshot (Chrome/Edge)
2. Click extension icon
3. Select "Capture visible part"
4. Save to screenshots folder
```

## 🎨 Screenshot Tips

### Do's ✅
- Use full browser window
- Clear, high resolution
- Show complete interface
- Include important text
- Good lighting (if screen photo)

### Don'ts ❌
- Don't crop important parts
- Don't include personal info
- Don't use low resolution
- Don't take blurry photos
- Don't include errors (unless showing fix)
